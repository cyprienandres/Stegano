## Getting started

### Requirements

* [Python 3](https://www.python.org/downloads/)
* [Tkinter module](https://wiki.python.org/moin/TkInter)

### How to run Stegano?

To run Stegano, execute this command from the root of the repository:
```
python3 stegano.py
```
